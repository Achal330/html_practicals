Tables & List

Unordered list
  • type="square" is display square bullets
  • type="circle" is display circle bullets
  • type="none" is no bullets.
  • In this style , the list items are not marked.
  • type="disc" is display bullets.
  • Disc is default style.
 The rowspan & colspan are "td" tag attributes.
The colspan attribute difines the number of columns in a table cell should span.
The rowspan attribute difines the number of row in a table cell should span.
Difinition List
dt tag Defines a term.
dd tag Describes the term.
